2018年のペパボ新卒採用サイトをつくったときの話

デザイン 採用 pepabo
  • ツイート

こんにちは！ホームページ作成サービス・グーペでデザイナーをしているちーちゃん(@chinu_t)です。はじめましての方に自己紹介をする際、自分をちゃん付けするのにとても抵抗があります。何か良い自己紹介の仕方があれば、ぜひ教えてください。そんなことはさておき、さっそく本題に入りましょう。

2018年度新卒向け採用サイトをつくりました

今回の新卒サイトのテーマは、総勢32名の勢いある歴代新卒が大集合！ 入社するまで〜入社後から今までに関するインタビューや、会社での一日の流れなどなど。ペパボの仲間の魅力をぎゅっと詰めたコンテンツ内容になりました。

（画像の情報は2016年サイト公開当時のものです）

新卒採用2018 GMOペパボ株式会社

やろうと思った経緯

ペパボの新卒サイトは、基本的にデザイナーが企画から制作まで担当しています。しかも社内公募なので、手を挙げた人が作ることができます。今回私が手を挙げた大きな理由としては、自分も新卒で入社をしたので、就活生に刺さるコンテンツが作れそう！と感じたからでした。当時新卒2年目、社内では就活生の気持ちが一番分かる立場にいるのではないかな？と考えたのです。

コンセプト作り

気持ちが一番分かる！とは思っていても、やはり自分の声だけでは頼りになりません。これまで新卒で入社した他の人たちや、その年の内定者にもヒアリングをしました。結果、ペパボの人や雰囲気を知りたいという声が多いことが分かりました。この声を形にするために、あえて新卒だけを出すコンテンツにするのはどうか？ という考えにいきつきました。キャリア採用の仲間は社会人暦や経験も豊富で、就活生にとって遠い存在なのでは？と考えたためです。

私たちの会社は新卒採用を開始して7年目。入社している人たちは6世代に渡ります。入社1年目の研修期間中の仲間から、マネージャーになっている先輩、キャリアはさまざま。入社したら、研修はこんなことをして、将来はこんなキャリアを歩むことができるキャリアパスを、コンテンツとして見せられるのでは？と考えました。ペパボが掲げる「もっと面白くできる」部分に関しては、とにかく数で勝負！32人を登場させることで他社との差別化を図ることにしました。

コンテンツ〜デザイン作成のこと

コンセプトを元に、どんなコンテンツにしていくか？UIは？サイトの雰囲気をつくるデザインは？…考えることは山のようにありました。しかし、かけられる時間や人数には限りがあります。今回は少人数かつ少ない時間での制作。短い期間で良いコンテンツにしていくには、トライ&エラーを重ねるしかない！ と考えました。

トライ&エラーを重ねるとは？

  • 考えを形にして、皆にぶつける。（=トライ）
  • ぶつけたものに、フィードバックをもらう。（=エラー）

もらったフィードバックを元に、また形にする…私の考えるトライ&エラーです。このフローをなるべく早く繰り返す。そうすることで、短期間で良いアイデア、デザインになっていきます。拙くても、人に見せたくなくても、目に触れるところに置いて、意見をもらう。自分の中で、「考えを笑われたらどうしよう…」「こんなアイデア恥ずかしくて出せない…」なんてモヤモヤ考えていても、前に進みません。恥ずかしい！という思いに打ち勝つこと。誰かに見せ、フィードバックをもらうことで、良い方向に進んでいきます。

トライ&エラーはこうやって重ねました

実際にトライ&エラーを重ねて、アイデアやデザインがリリースまでどうやって変わっていったのか？『Service x Design #4 未来を担う若手が語る、成長し続けるデザイナーとは？』でお話してきたときの資料に詳しく載せています。もしよかったら読んでみてください。

終わりに

トライ&エラーを重ねることで、目標の

  • 期限を守って無事リリース
  • 前年より応募者を増やす

以上二点を達成することができました。試行錯誤を重ねることは、今のデザイン業務にも日々生かされています。あの時失敗を重ねて上手く行ったのだから、今回も失敗も恐れずに行くぞ！の気持ちで、毎日頑張っています。みなさんも、失敗を恐れずトライ&エラー、ぜひ実践してみてください！
