「Design Scramble 2019」台風19号の接近に伴う開催中止のお知らせ

すでにお支払いいただいたチケット代金に関しましては、払い戻しをさせていただきます。 払い戻しに関する詳細は、peatixからお送りするメッセージをご確認ください。

Design Scramble（デザインスクランブル）とは？

「Design Scramble（デザインスクランブル）」は、渋谷を舞台としたクリエイター・デザイナーのためのデザインフェスティバルです。2018年よりはじまり、今年で2年目を迎えます。

「Design Scramble」の会場となるのは、参加企業のオフィス。IT・デザイン・ファッション・メディアアート・飲食…と、業界の垣根を超え繋がった様々な企業が企画した催し物を楽しみながら、1日を通してデザインに触れ合うことができます。

デザイナーのあなたにとって、そしてデザインに魅力を感じているあなたにとって、きっと特別な1日になるはずです。

繋がろう、渋谷。ここが、デザインの交差点。

GMOペパボ開催プログラム

12日 / 渋谷ストリーム4F

『スリぐるみ』がゲットできる？！「SUZURIのUFOキャッチャー」

12日（土）、渋谷ストリーム4Fの「SUZURI」ブースでは、「SUZURI」オリジナルのUFOキャッチャーを設置します。
「SUZURI」の公式忍者スリスリくんが手乗りサイズになったかわいいぬいぐるみ『スリぐるみ』がキャッチできる？！UFOキャッチャーが無料で遊べちゃいます！渋谷ストリームにぜひ遊びに来てください！

13日 / GMOペパボ (セルリアンタワー 11F)

今回初！「ペパボのサービスデザイン展」

13日（日）は、GMOインターネットグループ 11F（セルリアンタワー）での展示を行います！今回新しく展示するのは「ペパボのサービスデザイン展」。
GMOペパボはアイデアとテクノロジーで表現者を支えるための様々なサービスを展開しています。そこに生まれるユーザーとのタッチポイントはいわゆるデザイナーだけで作り上げるものではありません。あらゆる職種の活動の統合としてのサービス体験を提供するために、ペパボでは物事をどのように「デザイン」しているのか、その取り組みをご覧いただくことができます。

大好評につき再展！「ペパボのデザインプロセス展」

昨年来場者の方から好評だった「ペパボのデザインプロセス展」を再展示することになりました！
GMOペパボのサービスがどのようなプロセスでデザインされているのか、またどのように要望や課題をデザインで解決してきたのかをご覧いただけます。
昨年ご覧いただけなかった方も、もう一度見たいという方も、ぜひこの機会に足をお運びください！

【満席】トークイベント①「チームでデザインするための組織づくり」

情報テクノロジーが個を接続し社会の変化を加速させるこの時代、デザインの領域も個のクリエイティブからチームでの共創へと変革を求められています。組織横断のデザイン戦略に携わるデザイナー二人と取締役CTOとの三者で、テックとデザイン及び経営の視点を交えながら、GMOペパボの取り組みについてお話します。

小久保 浩大郎（GMOペパボ株式会社 / デザイングループマネージャー）
佐藤 咲（GMOペパボ株式会社 / プリンシパルデザイナー）

モデレーター
栗林 健太郎（GMOペパボ株式会社 / 取締役CTO）

【満席】トークイベント②「三者の視線の交点から浮かび上がるデザイン課題」

SaaS、toC、クライアントワークと、異なる業態の企業でデザインに関わる3人によるトークセッション。それぞれ違った立場にいながらも共通して感じている今取り組むべきデザイン課題をトピックとして取り上げ、その意味と背景、また現実的にどのように取り組むのかについて語り合います。

ガチャポンからでてくるのは…ドメイン？！「ドメインガチャ」

13日のGMOペパボ出展会場（GMOインターネットグループ11F（セルリアンタワー））では、なんとドメインがもらえるリアルなガチャポンを設置しています！
中身はトップレベルドメイン全6種。何が当たるかは出てくるまでのお楽しみです。ぜひ回してみてくださいね〜！

イベントの参加方法

1. チケット購入

「Design Scramble 2019」に参加するには、チケットの購入が必須です。
上記ボタンよりPeatixサイトへ移動し、「学生 / 一般」どちらかのチケットをご購入ください。

2. 各プログラム申し込み

トークセッションやワークショップなど、全体チケットの購入とは別に申込みが必要なプログラムがあります。
「要申し込み」と記載されたプログラムは、申込み手続きを忘れないようにしましょう！

3. イベント当日

当日、各会場の受付でPeatixのチケット購入画面をご提示ください。
イベント開催の2日間、全会場自由に入退場いただけます。
