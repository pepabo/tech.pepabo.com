ペパボのパートナーの自宅作業デスクまとめ

Pepabo
  • ツイート

こんにちは、HR統括グループのあちゃです。 GMOインターネットグループでは、新型コロナウィルスの感染拡大に備えて1月27日から在宅勤務を行っています。（詳細はこちらをご確認ください。）

27日の朝、自宅で業務を始めてすぐ@june29が、社内の GitHub Enterprise にこんなissueを作ってくれました。

ペパボにとって初めての長期間での在宅勤務を快適にするべく、自宅の作業デスクの工夫や知見がたくさん集まりました！ 今回はその中の一部を紹介したいと思います。

issueの発案者 @june29 のデスク

工夫ポイント

  • ケーブルはなるべく天板の下に追いやって、デスクの上をすっきりさせる
  • 色をなるべく黒で統一することでゴチャゴチャ感を軽減する

詳細は Scrapbox にまとめてあるそうなので、ぜひご覧ください！

毎日使うものにはお金をかけて実質無料にする方針 @hsbt のデスク

工夫ポイント

  • ディスプレイを横に二枚並べて空中にはみ出ない150cmのデスク
  • ディスプレイは32インチを二枚
  • マシンは左の棚に全部入れて作業時に目の前には見えないように。充電関連も全部左の棚。
  • ビデオチャットは iPad Pro か Pixel 3 でやる
  • 椅子はアーロンチェア

とにかくモノを置きたくない！ @akichan のデスク

工夫ポイント

  • デスクと椅子は無印良品のものでシンプルに
    • 奥行70cmのデスクで広々と
  • 机右手前に設置したコンセントを設置
  • ACアダプタ等はデスクにぶら下げたカゴの中に収納

スタンドデスクで健康的に @tora のデスク

工夫ポイント

  • 足元を Dyson Pure Hot + Cool Link で温めている
  • スタンドデスクだと足の動きがあるとだいぶ違うのでヨガブロックに乗ったり降りたりしている
  • スタンドデスク・ライト・マット・ワゴンは全部 IKEA

写っている猫は仕込んでいないそう。在宅勤務、ペットとの触れ合いもメリットの１つかも！

【番外編】ライフハックデスク

今回は突然の在宅勤務だったためデスクの準備ができず、家にあるものをデスク代わりにしたパートナーもいました。

高さの調整が自由自在！ @otayu のデスク

おすすめポイント

  • 二個重ねれば高さ調節も可能。抜群の機能性。
  • クローゼットケースと机は紙一重。

その後、無事にデスクを購入できたそうです！

キッチンカウンターをデスクに！ @yuta25 のデスク

おすすめポイント

  • 高さがちょうど良いのと、飲み物がすぐ調達できるのが良いです
